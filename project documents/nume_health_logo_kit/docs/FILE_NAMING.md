# File naming convention

Recommended pattern:

`{brand}_{asset}_{variant}_{color}_{bg}_{size}.{ext}`

Examples:
- `numehealth_lockup_primary_color_transparent_w4096.png`
- `numehealth_icon_mark_white_transparent_1024.png`
- `numehealth_social_og_dark_1200x630.png`
- `numehealth_app_icon_gradient_1024.png`

Keep:
- `lockup` for logo + wordmark
- `icon` or `mark` for the symbol only
- `color`, `white`, `black` for colorways
- `transparent`, `light`, `dark` for background context
